# Agência Criativa Web

Landing page responsiva desenvolvida como projeto de estudo do curso Full Stack JavaScript da EBAC.

## Tecnologias

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

## Funcionalidades

- Layout responsivo
- Menu de navegação
- Banner principal
- Seção Sobre
- Serviços
- Depoimentos
- Formulário de contato

## Responsividade

O projeto foi desenvolvido para funcionar em:

- Desktop
- Tablet
- Smartphone
